解説
「次の 2 つの条件式の値が True であるか」は、 and 演算子を用いて確かめることができます。

A <= B and B < C
なお、Python では同等の条件式を次のように簡潔に書くことができます。

A <= B < C
しかし、この記法は直感的ではあるもののプログラミングの記法としては特殊なため、型や条件式などの基本に慣れるまではできるだけ使わないほうがよいでしょう。

解答例
# 標準入力から値を受け取る
# A, B, C: int 型
A = int(input())
B = int(input())
C = int(input())

if A <= B and B < C: # もし B が A 以上かつ C 未満なら
    print("Yes")
else: # そうでないなら
    print("No")
